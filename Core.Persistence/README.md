# Ba.Core.Persistence

A modern and extensible Entity Framework Core–based Persistence Library designed for scalable, clean, and maintainable .NET applications.

## Features

- Generic Repository Pattern
- Unit of Work support
- Cascading Soft Delete
- Paging utilities (ToPaginate, Paginate, PaginateAsync)
- Dynamic Querying (Filter, Sort, Composite Filtering)
- Include support for deep navigation loading
- Async CRUD operations
- Fully compatible with Entity Framework Core 10

---

## Installation

Install via .NET CLI: 
```bash
dotnet add package Ba.Core.Persistence

Or via NuGet Package Manager:
```powershell
Install-Package Ba.Core.Persistence
